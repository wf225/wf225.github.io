## 本地搭建

确保已安装`Ruby 2.1.0` 或更高版本：

```sh
ruby --version
```

安装`Bundler`：

```sh
gem install bundler
```

安装依赖：

```sh
npm install
```

运行 Jekyll：

```sh
npm start
```
